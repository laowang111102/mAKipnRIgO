# 人事管理系统 java647

## 项目概述

人事管理系统是一款基于Springboot+Vue的前后端分离的Javaee项目。该项目采用MVC三层体系结构，整合Spring、SpringBoot、MyBatis和Vue等技术开发，以MySQL数据库作为数据存储，实现了B/S架构的Web系统。该系统界面简洁，操作方便。

## 技术栈

- 前端：Vue
- 后端：Springboot
- 数据库：MySQL
- 项目管理工具：Maven

## 功能模块

- 用户管理
- 员工信息管理
- 薪资管理
- 考勤管理
- 招聘管理

## 系统角色

- 系统管理员
- 人事专员
- 普通员工

## 运行环境

- 操作系统：Windows、Linux或MacOS
- 数据库：MySQL 5.7+
- JDK版本：1.8+

## 部署步骤

1. 准备MySQL数据库环境
2. 导入项目数据库
3. 编译后端代码，部署到Java应用服务器（如Tomcat）
4. 前端代码编译并部署到Web服务器（如Nginx）
